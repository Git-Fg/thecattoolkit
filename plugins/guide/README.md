# guide

**Prompt Engineering as Guidance** - Steering agents through structured prompts.

## Purpose

Focuses on steering other agents through well-crafted prompts. Positions prompt engineering as a guidance mechanism rather than just a creation tool.

## Skills

- **prompt-engineering** - Prompt design theory (CoT, few-shot), XML vs Markdown decision framework
- **prompt-library** - Template library for single prompts, chains, and meta-prompts

## Agents

- **prompt-engineer** - Designs and optimizes prompts for production use

## Usage

Skills are user-invocable with `context: fork` - invoke `/prompt-engineering` or `/prompt-library` directly.

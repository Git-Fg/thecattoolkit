# Notes Template

Complete template for `notes.md` for storing research findings and intermediate work.

## Basic Notes Template

```markdown
# Notes: [Topic]

## Sources
### Source 1: [Name]
- URL: [link]
- Key points:
  - [Point 1]
  - [Point 2]
  - [Point 3]
- Date accessed: [Date]
- Relevance: [High/Medium/Low]

### Source 2: [Name]
- URL: [link]
- Key points:
  - [Point 1]
  - [Point 2]
  - [Point 3]
- Date accessed: [Date]
- Relevance: [High/Medium/Low]

## Research Findings
- [Finding 1]
- [Finding 2]
- [Finding 3]

## Key Insights
- [Insight 1]
- [Insight 2]
- [Insight 3]

## Questions Raised
- [Question 1]
- [Question 2]
- [Question 3]

## Data Points
| Metric | Value | Source |
|--------|-------|--------|
| [Metric 1] | [Value] | [Source] |
| [Metric 2] | [Value] | [Source] |

## Quotes
> "[Relevant quote from source]"
> —[Author], [Source]

> "[Relevant quote from source]"
> —[Author], [Source]

## Action Items
- [ ] [Action item 1]
- [ ] [Action item 2]
- [ ] [Action item 3]
```

# Task Plan Template

Complete template for `task_plan.md` with all sections.

## Basic Template

```markdown
# Task Plan: [Task Name]

## Goal
[One sentence describing the desired end state]

## Phases
- [ ] Phase 1: [Name]
- [ ] Phase 2: [Name]
- [ ] Phase 3: [Name]
- [ ] Phase 4: [Name]
- [ ] Phase 5: [Name]

## Key Questions
1. [Question to answer]
2. [Another question]
3. [Third question]

## Decisions Made
- [Timestamp] [Decision]: [Rationale]
- [Timestamp] [Decision]: [Rationale]

## Errors Encountered
- [Timestamp] [Error]: [Resolution]
- [Timestamp] [Error]: [Resolution]

## Status
**Currently in Phase X** - [Current focus]
**Next:** [What comes next]
**ETA:** [Time estimate]
```

```markdown
- [ ] Phase 5: Create deliverable

2. [Research question]
3. [Research question]

- [Count] sources collected

## Research Streams (Optional)

- Stream B: [Topic] - [Status]

```

# Agent Best Practices Command

Invoke the agent-best-practices skill to get comprehensive guidance on building effective AI agents.

## Usage

### Get General Overview
```
/sys-cognition:agent-best-practices
```
Provides an overview of agent development best practices and core principles.

### Get Specific Topic
```
/sys-cognition:agent-best-practices planning
```
Gets detailed guidance on a specific topic. Available topics:
- `planning` - Plan-first approach and planning workflows
- `patterns` - Implementation patterns and techniques
- `anti-patterns` - What to avoid and common mistakes
- `workflows` - Ready-to-use workflow templates
- `templates` - Reusable templates and checklists

### Example Workflow
```
"Help me plan a complex feature using agent best practices"
-> Guides you through the planning-first approach

"I need to fix a bug - what workflow should I use?"
-> Recommends the bug fix workflow template

"What are common anti-patterns in agent development?"
-> Provides anti-patterns guidance with examples

"I need a code review checklist"
-> Provides the comprehensive review checklist
```

## What You'll Get

1. **Core Principles** - Fundamental concepts for agent development
2. **Implementation Patterns** - Detailed patterns with examples
3. **Workflow Templates** - Ready-to-use workflows for common scenarios
4. **Anti-Patterns** - What to avoid with explanations
5. **Reusable Assets** - Templates, checklists, and guidelines

The skill provides progressive disclosure - start with the overview and dive deeper into specific topics as needed.

# Claude Code Mastery Command

Invoke the Claude Code mastery skill to access comprehensive guides for mastering Claude Code.

## Usage

Invoke Skill(claude-code-mastery) to access:

**Default**: Shows quick start guide and prompts for topic selection
- Fundamentals (plan mode, CLAUDE.md, context, prompting)
- Advanced Features (skills, subagents, MCP)
- Context Optimization (degradation, management)
- Patterns & Best Practices (proven workflows)

**With specific topic**:
- `/sys-cognition:claude-code-mastery fundamentals` - Quick start guide
- `/sys-cognition:claude-code-mastery advanced` - Skills, subagents, MCP
- `/sys-cognition:claude-code-mastery context` - Context window strategies
- `/sys-cognition:claude-code-mastery patterns` - Best practices and anti-patterns

# Question Bank

Contextual questions for intake, organized by purpose.

## Universal Questions

### Topic Identifier
When topic not obvious:

**Question:** What topic/feature is this for? (used for file naming)

Let user provide via "Other" option. Enforce kebab-case.

### Chain Reference
When existing files found:

**Question:** Should this prompt reference any existing research or plans?

Options: List found files + "None"

## Do Questions

### Artifact Type
**Question:** What are you creating?

Options:
- Code/feature - Software implementation
- Document/content - Written material
- Design/spec - Architecture, specifications

### Scope
**Question:** What level of completeness?

Options:
- Production-ready - Ship to users
- Working prototype - Functional
- Proof of concept - Minimal demonstration

### Testing
**Question:** What testing is needed?

Options:
- Full coverage - Unit, integration, e2e
- Core functionality - Key paths tested
- Manual verification - No automated tests

## Plan Questions

### Plan Purpose
**Question:** What is this plan leading to?

Options:
- Implementation - How to build
- Decision - Weigh options, choose approach
- Process - Define workflow

### Format
**Question:** What format works best?

Options:
- Phased roadmap - Sequential stages
- Checklist/tasks - Actionable items
- Decision framework - Criteria, trade-offs

## Research Questions

### Depth
**Question:** How deep should research go?

Options:
- Overview - High-level understanding
- Comprehensive - Detailed exploration
- Exhaustive - Everything available

### Sources
**Question:** What sources to prioritize?

Options:
- Official docs - Primary sources
- Community - Real-world examples
- Current/latest - 2024-2026 sources

### Focus
**Question:** What aspect is most important?

Options:
- How it works - Concepts, internals
- How to use it - Patterns, examples
- Trade-offs - Pros/cons, alternatives

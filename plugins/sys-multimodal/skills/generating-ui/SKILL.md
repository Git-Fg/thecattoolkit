# Canvas Design

Create beautiful visual art through a two-step philosophy-driven design process, producing museum-quality original artwork as .png or .pdf documents.

## When to Use This Skill

- Creating posters or promotional artwork
- Designing original visual art pieces
- Developing brand visuals or identity elements
- Creating artistic compositions or abstractions
- Producing visual content for presentations or displays
- Generating design-forward artwork with deep conceptual foundations

## Core Philosophy

This skill operates on a **two-step process** that transforms subtle conceptual references into masterful visual expressions:

### Step 1: Design Philosophy Creation

Create a **VISUAL PHILOSOPHY** (aesthetic movement manifesto) that serves as the foundation for all visual decisions. This is expressed as a .md file containing:

**Movement Naming** (1-2 words):
- Examples: "Brutalist Joy", "Chromatic Silence", "Metabolist Dreams"

**Philosophical Articulation** (4-6 paragraphs):
The philosophy must emphasize how the aesthetic manifests through:
- **Space and form**: Spatial relationships and sculptural qualities
- **Color and material**: Chromatic systems and tactile representation
- **Scale and rhythm**: Proportional harmony and visual tempo
- **Composition and balance**: Visual equilibrium and tension
- **Visual hierarchy**: Information flow through design elements

**Critical Guidelines:**
- **Avoid redundancy**: Each design aspect mentioned once with depth
- **Emphasize craftsmanship**: Stress that work should appear meticulously crafted, the product of countless hours by a master
- **Leave creative space**: Be specific enough to guide, open enough for interpretation
- **Visual expression priority**: Ideas communicated through form, not text

**Example Philosophy Structures:**

*Concrete Poetry*: Communication through monumental form and bold geometry. Visual expression through massive color blocks, sculptural typography, Brutalist spatial divisions. Ideas expressed through visual weight and spatial tension, not explanation. Text as rare, powerful gesture—never paragraphs, only essential words integrated into visual architecture.

*Chromatic Language*: Color as primary information system. Geometric precision where color zones create meaning. Typography minimal—small sans-serif labels letting chromatic fields communicate. Information encoded spatially and chromatically. Words only to anchor what color already shows.

*Analog Meditation*: Quiet visual contemplation through texture and breathing room. Paper grain, ink bleeds, vast negative space. Photography and illustration dominate. Japanese photobook aesthetic. Images breathe across pages. Text whispered sparingly—short phrases, never explanatory blocks.

### Step 2: Canvas Creation

Express the design philosophy visually, creating original artwork that embodies the aesthetic principles. This is a **single-page, highly visual composition** produced as .png or .pdf.

**Visual Creation Principles:**

**Form and Space:**
- Use form, space, color, and composition as primary communication tools
- Create visual hierarchies that guide the eye naturally
- Employ spatial relationships to convey meaning
- Balance positive and negative space with intention

**Color and Material:**
- Develop cohesive color palettes that support the philosophy
- Use color strategically to create mood, depth, and meaning
- Consider material qualities even in digital medium (texture, opacity, layering)

**Scale and Rhythm:**
- Create visual tempo through element sizing and repetition
- Use scale contrasts to establish importance and flow
- Establish rhythmic patterns that unite the composition

**Typography (Minimal, Essential):**
- Text always minimal and contextually appropriate
- Font selection must be design-forward and visually integrated
- Typography serves the visual communication, never dominates
- Use different fonts strategically (search `./canvas-fonts` directory)
- Ensure perfect spacing, alignment, and readability
- Text never overlaps, nothing falls off canvas boundaries
- Maintain proper margins and breathing room

**Museum-Quality Craftsmanship:**
- Work must appear meticulously crafted through countless refinements
- Every element placed with precision of master craftsman
- Composition, spacing, and color choices demonstrate expert-level skill
- Result should be undeniably impressive and display-worthy
- Avoid amateur or cartoony aesthetics—maintain sophistication

**Reference Subtleties:**
- Weave subtle conceptual threads invisibly into form and color
- Create sophisticated references that enhance depth without announcing themselves
- Approach like jazz musician quoting another song—only those who know will catch it
- Everyone should appreciate the artistry even without catching the reference

## File Outputs

This skill produces three types of files:

1. **Design Philosophy (.md)**: Written manifesto defining the aesthetic movement
2. **Visual Artwork (.png)**: High-resolution raster artwork
3. **Visual Artwork (.pdf)**: Vector-based or high-quality artwork suitable for print

## Design Execution Process

### 1. Deduce the Subtle Reference

**Critical Step**: Before creating the canvas, identify the conceptual framework from the request. The topic is a **subtle, niche reference embedded within the art**—not always literal, always sophisticated. Someone familiar should feel it intuitively; others experience masterful abstract composition.

The design philosophy provides the aesthetic language. The conceptual framework provides the soul—the quiet conceptual DNA woven invisibly into form, color, and composition.

### 2. Canvas Creation Process

**Mental Preparation**: Clear the mind and gather thoughts before beginning visual creation.

**Approach**: Use the design philosophy as foundation to craft a masterpiece. Create **one single page, highly visual, design-forward output** unless multiple pages requested.

**Visual Language**: Treat the abstract philosophical design as a scientific observation—dense accumulation of marks, repeated elements, or layered patterns building meaning through patient repetition. Add sparse, clinical typography suggesting diagrams from an imaginary discipline.

**Composition Techniques**:
- Use repeating patterns and perfect geometric shapes
- Embrace systematic visual language for expressing human experience
- Create artifacts proving ephemeral concepts can be studied and mapped
- Anchor with simple phrases positioned subtly
- Use limited, intentional color palettes

**Text Integration**:
- Minimal, contextual, always visual-first
- Could be whisper-quiet labels or bold typographic gestures
- Context guides scale (punk venue poster might have larger, aggressive type)
- Font should generally be thin and design-forward
- Typography becomes part of the art itself—bring fonts onto canvas

**Technical Execution**:
- Search and download fonts from `./canvas-fonts` directory as needed
- Make typography part of the art, not typeset digitally
- Embrace ultimate design freedom and aesthetic choice
- Push boundaries and aesthetics to frontier
- Follow design instinct guided by philosophy

### 3. Refinement Phase

**Quality Standard**: Work must be pristine, a masterpiece of craftsmanship, display-worthy.

**Refinement Approach**: Don't add more graphics—refine what exists. Make it extremely crisp while respecting design philosophy and minimalist principles.

**Second Pass Process**:
- Ask: "How can I make what's already here more of a piece of art?"
- Avoid filters or font refactoring
- Make existing composition more cohesive
- Polish details rather than adding elements
- Double-check everything is perfect

## Multi-Page Options

When multiple pages requested, create additional creative pages following the same philosophy but distinctly different. Bundle in single .pdf or multiple .png files.

**Page Development**:
- First page as single page in coffee table book
- Next pages as unique twists and interpretations
- Tell story tastefully through sequential compositions
- Exercise full creative freedom
- Maintain philosophical coherence while exploring variations

## Essential Principles

- **VISUAL PHILOSOPHY FIRST**: Create aesthetic worldview before making
- **MINIMAL TEXT**: Always sparse, essential-only, integrated as visual element
- **SPATIAL EXPRESSION**: Ideas communicate through space, form, color—never paragraphs
- **ARTISTIC FREEDOM**: Interpret philosophy visually with creative room
- **PURE DESIGN**: Make ART OBJECTS, not decorated documents
- **EXPERT CRAFTSMANSHIP**: Emphasize meticulous execution repeatedly

## Copyright Compliance

**CRITICAL**: Create only original designs. Never copy existing artists' work. This skill produces entirely original artwork inspired by philosophical frameworks, not existing pieces. All visual expression must be created from scratch, ensuring copyright safety and artistic integrity.

## Technical Requirements

**Output Specifications**:
- High-resolution artwork suitable for display or print
- Proper margins and canvas boundaries
- Professional typography with perfect spacing
- Cohesive color palettes
- Museum-quality composition and execution

**Font Resources**:
- Search `./canvas-fonts` directory for font options
- Download additional fonts as needed for sophistication
- Ensure font choices support the design philosophy
- Make typography design-forward and visually integrated

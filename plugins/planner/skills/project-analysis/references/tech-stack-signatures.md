# Tech Stack Signatures

## Package Managers
- `package.json` -> Node.js / TypeScript / JavaScript / Bun
- `requirements.txt`, `Pipfile`, `pyproject.toml` -> Python / Uv
- `go.mod` -> Go
- `Cargo.toml` -> Rust
- `pom.xml` -> Java (Maven)
- `build.gradle` -> Java/Kotlin (Gradle)
- `Gemfile` -> Ruby
- `composer.json` -> PHP

## Web Frameworks
- **React**: `react`, `react-dom`
- **Next.js**: `next`, `pages/` or `app/` directory
- **Vue**: `vue`, `.vue` files
- **Express**: `express` dependency
- **FastAPI**: `fastapi`, `uvicorn`
- **Django**: `django`, `manage.py`
- **Flask**: `flask`, `app.py`
- **Rails**: `rails`, `app/models`, `app/controllers`
- **Spring Boot**: `@SpringBootApplication` annotations

## Infrastructure
- **Docker**: `Dockerfile`, `.dockerignore`
- **Kubernetes**: `k8s/`, `charts/`, `deployment.yaml`
- **Terraform**: `*.tf` files
- **Serverless**: `serverless.yml`, `netlify.toml`, `vercel.json`
- **GitHub Actions**: `.github/workflows/`

## Testing
- **Jest**: `jest.config.js`, `*.test.js`
- **Pytest**: `pytest.ini`, `tests/test_*.py`
- **Go Testing**: `*_test.go` files
- **Cypress/Playwright**: `cypress.json`, `playwright.config.ts`

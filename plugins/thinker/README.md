# @cattoolkit/thinker

Consolidated reasoning and prompt engineering plugin (formerly strategist, prompter, think).

## Skills

- **thinking-frameworks** - Structured thinking frameworks for analysis, decision-making, prioritization
- **prompt-engineering** - Prompt design patterns, optimization, anti-patterns
- **prompt-library** - Template library for single prompts, chains, and meta-prompts

## Agents

- **brainstormer** - Strategic analysis using thinking frameworks
- **prompt-engineer** - Prompt design and optimization specialist

## Design Philosophy

All skills are **user-invocable** with `context: fork` for isolated execution. No command wrappers needed - invoke skills directly via `/thinking-frameworks` or `/prompt-engineering`.
